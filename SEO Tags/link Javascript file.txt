#LINK JAVASCRIPT FILE IN HTML PAGE

-Add Javascript file to HTMl page.
-For this you need to use script tag in head tag.
-In script tag write your javacript.
-For add Javascript file use script tag and src attribut for giving file path.

EX:-
      <!DOCTYPE html>
<html>
    <head>
        <title>link javascrip file</title>
       <!-- <script>
            alert("hello");
        </script>   -->
        <script src="javascript.js"></script>
    </head>
    <body>
        
    </body>
</html>
